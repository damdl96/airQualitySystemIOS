# Air Quality Dashboard on IOS
  
Materials:

- iPhone XR.
- NODEMCU V 1.0
- DHT 11 (For Temperature and humidity)
- CCS811 (For TVOc's and CO2)

App developed for IOS 12.2

## Documentation 

- [Arduino Firebase](https://firebase-arduino.readthedocs.io/en/latest/)
